# Mockup API

The commands are run from the mockup APIs root folder, where `server.js` and `package.json` are located.

## Development

## Setup

```
npm install
```

In `./config.js`, `IMAGES_PATH` must be set to the public image folder.

## Run

```
npm run dev
```

## Run tests

```
npm test .\tests
```

## Run individual test

```
npm test .\tests\<filename>.js
```

## Normal

### Build

```
npm build
```

### Run

```
npm start
```
